# Curso de Lógica de Programação

Estas são minhas anotações do curso Algorítmos e Lógica de Programação do professor Nélio Alves na Udemy.

O curso de 31 horas aborda conceitos básicos de programação como variáveis, tipos de dados, operadores aritméticos, operadores comparativos, operadores lógicos, saída de dados, entrada de dados, estruturas
condicionais, estruturas repetitivas e arrays.

O curso se inicia utilizando a linguagem de aprendizado Portugol, mas após isso também ensina a lógica básica de funcionamento de Python, C, C++, C# e Java com aulas padrão e exercícios com correção.

Ele não se aprofunda em nenhuma linguagem ou ensina os fundamentos, como por exemplo como o cmd lê os textos escritos quando aplicado um comando de leitura, ou como números de ponto flutuante são armazenados no
sistema. Esse é um curso puramente de lógica de programação, mas ainda assim, ele é muito útil para treinar seu pensamento computacional e atiçar sua curiosidade.

A ordem de leitura deste repositório é Portugol -> Python -> C -> C++ -> Java -> C# (lendo primeiramente as anotações e só então as atividades)
Algumas aulas mais próximas do final do curso terão menos anotações e anotações mais complexas, isso se dá pela estrutura semelhante das linguagens que reduz a necessidade de repetir as mesmas anotações diversas vezes
e pelo meu crescente interesse sobre os fundamentos da programação respectivamente.
